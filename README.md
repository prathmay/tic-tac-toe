# Tic-Tac-Toe

A small browser-based Tic-Tac-Toe game implemented with plain HTML, CSS, and JavaScript.

## Features
- Two-player local gameplay (X and O)
- Clean, responsive UI

## Files
- `index.html` — game UI and markup
- `index.css` — styles
- `index.js` — game logic

## Run
Open `index.html` in your web browser (double-click the file or serve the folder). No build step or server required.

## Development
- Edit `index.js` to modify game logic.
- Edit `index.css` to change styling.
